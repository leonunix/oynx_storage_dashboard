Onyx Brand Study

三个方向都做成成品预览了，直接挑气质最对的那个。

每个方案都同时展示了大尺寸主视觉、横向 lockup 感觉，以及小尺寸图标缩放效果。 如果你挑中一个，我下一步就把它正式接进 dashboard 侧栏、登录页和 favicon。

Facet O

黑曜石切面组成抽象 O，强调产品名和高识别度，最稳也最适合 favicon。

Concept 1
ONYX Storage Dashboard
高级感强
辨识度稳定
最适合图标化
  • 更像基础设施产品品牌，不会落入“硬盘图标”老套路。
  • 和现在 dashboard 的蓝绿主色最容易融合，改造成本最低。

Monolith

把 Onyx 当成一块发光切缝的矿石立柱来表现，技术感和“存储引擎味”最强。

Concept 2
ONYX Engine Control
硬核感最强
最像存储设备
存在感很高
  • 如果你想让产品气质更“内核工程”“写路径”“数据通道”，这个方向最直接。
  • 缩到极小尺寸时不如 Facet O 那么稳，但在登录页和海报位很有冲击力。

Core Ring

中心核心加外部数据环，偏控制平面、观测与编排，整体更企业化和现代化。

Concept 3
ONYX Control Plane
控制平面感强
企业级最稳重
适合 dashboard
  • 更偏“调度中心 / 观测中心”的品牌感，适合做后台系统和登录页品牌头图。
  • 图标层面也不错，但会比 Facet O 稍微更抽象一点。
